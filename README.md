# ✨ lanvalird — мой маленький цифровой сад ✨

> Контент в этом файле изначально был подготовлен только для frontend-части приложения. Будь внимателен.

Привет! Это не просто репозиторий, это дом для моего персонального сайта-портфолио. Здесь, в этом уютном уголке интернета, собраны все мои проекты, навыки и немного души. Добро пожаловать! ☕

Сайт написан на **Next.js 16** и причёсан с помощью **Tailwind CSS**. Если тебе интересно, как это всё работает внутри — милости прошу под кат.

## 🚀 Что тут внутри?

- **Next.js 16 (App Router)**: Серверные компоненты и статическая генерация для скорости и удобства.
- **React 19**: Живые и отзывчивые интерфейсы.
- **Tailwind CSS 4**: Стилизация, от которой не болят глаза.
- **TypeScript**: Порядок в типах и спокойствие в душе.
- **MDX**: Потому что контент для проектов должен быть гибким.
- **Hono**: Лёгкий API на плечах Vercel.
- **Radix UI** & **Lucide React**: Красивые и доступные компоненты с иконками.
- **next-themes**: Переключаемся между светлым и тёмным, как настроение.
- **Biome**: Всё, что нужно для чистоты кода.
- **pnpm**: Быстро и экономно.

## 📦 Что умеет сайт?

- **Главная страница**: Приветствие, аватарка (с магическим свечением) и кнопка, чтобы сразу нырнуть в мои навыки.
- **Навыки**: Разложены по полочкам: Fullstack, Frontend, Backend, Mobile, Native и даже творческие хобби (дизайн, музыка). Карточки с бейджиками, всё как я люблю.
- **Проекты**: Сетка с карточками всех моих творений. Можно посмотреть детали, почитать описание и потыкать ссылки.
- **Страница проекта**: Тут живёт полная история: MDX-контент, красивая обложка на фоне, теги, даты и ссылки на GitHub/демо.
- **Страница ссылок**: Все мои соцсети и профили в одном месте.
- **API** (Hono):
  - `GET /api/projects` — получить список проектов (с пагинацией и фильтром по slug).
  - `GET /api/skills` — получить список навыков.
- **Тёмная/светлая тема**: Встроено и работает.
- **Server Actions**: Кнопка, которая заставит сайт перезагрузить данные о проектах, если что-то пошло не так.
- **Open Graph & Favicon**: Динамические иконки для проектов, чтобы делиться было красиво.

## 📁 Как это всё устроено?

```
src/
├── app/                          # Сердце приложения
│   ├── (home)/                   # Главная страница
│   ├── projects/                 # Список и детальные страницы проектов
│   ├── skills/                   # Страница навыков
│   ├── links/                    # Страница со ссылками
│   ├── api/                      # Hono API
│   ├── layout.tsx                # Корневой layout (темы, хедер, фон)
│   ├── not-found.tsx             # Страница 404 (с юмором)
│   └── globals.css               # Глобальные стили и CSS-переменные
├── shared/
│   ├── components/ui/            # Кнопки, карточки, бейджики...
│   ├── config/                   # Ссылки на соцсети
│   ├── data-storage/             # Локальные данные (проекты, навыки)
│   ├── lib/                      # Вспомогательные функции (cn, api)
│   └── types/                    # TypeScript-типы
├── widgets/
│   └── app-header/               # Навигация
└── mdx-components.tsx            # Стилизация MDX
```

## 🛠️ Как запустить?

1.  **Склонируй репозиторий:**

    ```bash
    git clone https://github.com/lanvalird/lanvalird-website.git
    cd lanvalird-website
    ```

2.  **Установи зависимости:** (Мы используем `pnpm`, потому что он быстрый и бережливый)

    ```bash
    pnpm install
    ```

3.  **Запусти режим разработки:**

    ```bash
    pnpm dev
    ```

    Открой [http://localhost:3000](http://localhost:3000) и любуйся.

4.  **Собери продакшн:**

    ```bash
    pnpm build
    pnpm start
    ```

5.  **Причеши код:**
    ```bash
    pnpm format
    ```

## 📝 Работа с данными

- **Навыки (`shared/data-storage/skills/`)**: Данные хранятся в TypeScript-файлах. Есть удобные хелперы `addCategoryForEach` и `addCategoriesForEach`, чтобы не повторяться.
- **Проекты (`shared/data-storage/projects/index.ts`)**: Массив объектов `Project`. Каждый проект может содержать `slug`, `name`, `description`, `markdown`-строку, ссылки, картинки, теги и даты.
- **Конфигурация (`shared/config/social-links.ts`)**: Просто массив объектов со ссылками на все мои профили.

## ✨ Что делает этот сайт особенным?

- **Статическая генерация**: Страницы проектов генерируются заранее, поэтому летают.
- **Адаптивность**: Всё красиво смотрится и на большом мониторе, и на телефоне.
- **Интереснный фон**: Карточки проектов и навыков имеют приятный радиальный градиент.
- **Аватар с душой**: Изображение не просто висит, а имеет размытую копию на фоне. Магия CSS.
- **MDX-контент**: Можно писать сложные штуки прямо в описании проекта.
