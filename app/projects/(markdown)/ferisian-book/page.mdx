# Ferisian Book

Учебник был создан для реализации онлайн-справочника по ферийскому языку, а также обновлению и совершенствованию креативных навыков его создателя (меня).

Требования:

- Использовать React
- Изучить фреймворк для создания документаций
- Мультиязычность
- Удобство
